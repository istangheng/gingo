# Gingo

Gin Web 项目脚手架

## 项目结构

参考 [Standard Go Project Layout](https://github.com/golang-standards/project-layout) 目录结构

- api: 接口
- cmd： 程序启动入口
- configs： 配置文件
- deployments： 部署文件
- initialize： 初始化
- internal： 内部包
- middleware： 中间件
- model： 模型
- router： 路由
- service： 服务层
- test： 测试包
- third_party： 三方件
- tools： 工具包
- vendor： 依赖包
